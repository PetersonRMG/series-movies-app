# 🎬 App de Filmes e Séries

Aplicação web desenvolvida em **React + Vite** para exibição de filmes e séries, com uma interface moderna e responsiva.

---

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS / Bootstrap
- API de filmes/séries (TMDB)

 

---

## ⚙️ Funcionalidades

- Listagem de filmes
- Listagem de séries
- Interface separada por categorias
- Layout responsivo
- Navegação simples e intuitiva

---

## 🛠️ Como rodar o projeto localmente

### Pré-requisitos
- Node.js instalado (versão 18  )
- Yarn instalado

### Passos

```bash
# Clone o repositório
git clone https://github.com/PetersonRMG/series-movies-app.git

# Entre na pasta
cd series-movies-app

# Instale as dependências
yarn install

# Rode o projeto
yarn dev
